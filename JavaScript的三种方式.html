JavaScript的三种使用方式

1. 在<head>标签中使用<script>标签

这种方式将JavaScript代码放在HTML文档的<head>部分。

优点：代码集中管理，可以在页面加载前定义函数和变量。

缺点：如果脚本较大，可能会延迟页面的加载和渲染。

当前时间：点击按钮显示
<head>
    <script>
        function showCurrentTime() {
            const now = new Date();
            document.getElementById('current-time').textContent = now.toLocaleTimeString();
        }
    </script>
</head>

2. 在<body>标签中使用<script>标签

这种方式将JavaScript代码放在HTML文档的<body>部分，通常在页面内容的底部。

优点：不会阻塞页面渲染，可以在页面加载后立即执行。

缺点：如果在页面顶部引用，可能会找不到尚未加载的DOM元素。

点击按钮改变此区域的背景颜色
<body>
    <!-- 页面内容 -->
    
    <script>
        document.getElementById('change-color-btn').addEventListener('click', function() {
            const result = document.getElementById('color-result');
            const randomColor = '#' + Math.floor(Math.random()*16777215).toString(16);
            result.style.backgroundColor = randomColor;
            result.textContent = '背景颜色已更改为: ' + randomColor;
        });
    </script>
</body>

3. 引用外部JavaScript文件

这种方式通过<script>标签的src属性引用外部的.js文件。

优点：代码可重用，便于维护，可以被浏览器缓存，减少加载时间。

缺点：需要额外的HTTP请求来获取文件。

= ? 
<script src="calculator.js"></script>

外部文件 calculator.js 的内容：

document.getElementById('calculate-btn').addEventListener('click', function() {
    const num1 = parseFloat(document.getElementById('num1').value);
    const num2 = parseFloat(document.getElementById('num2').value);
    const operator = document.getElementById('operator').value;
    let result;
    
    switch(operator) {
        case '+': result = num1 + num2; break;
        case '-': result = num1 - num2; break;
        case '*': result = num1 * num2; break;
        case '/': result = num1 / num2; break;
    }
    
    document.getElementById('result').textContent = isNaN(result) ? '请输入有效数字' : result;
});

总结

JavaScript的三种使用方式各有优缺点：

  • 在<head>中使用：适合定义全局函数和变量，但可能延迟页面加载。
  • 在<body>底部使用：不会阻塞页面渲染，适合操作DOM元素。
  • 引用外部文件：代码可重用，便于维护，可以被缓存，是最推荐的方式。

在实际开发中，通常会结合使用这三种方式，根据具体需求选择最合适的方式。
